# LinkedIn Clone — Frontend

A full-stack LinkedIn clone built with Next.js and Tailwind CSS.

## Tech Stack

- **Framework:** Next.js 16
- **Styling:** Tailwind CSS
- **State Management:** Zustand
- **HTTP Client:** Axios

## Features

- User Authentication (Register / Login / Logout)
- Home Feed — Create posts, Like, Comment
- Network Page — Send / Accept / Remove connection requests
- Messaging — Chat with connections
- Notifications — Connection requests, Likes, Comments
- Jobs — Post jobs, Apply for jobs
- Profile — Edit name, headline, profile picture

## Getting Started

1. Clone the repo
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create `.env.local` file:
   ```
   NEXT_PUBLIC_API_URL=http://localhost:8000/api
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
